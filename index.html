Sticky. A simple notification system for jQuery

Demo

Start with some basic messages. The default position is top-right with an 'autoclose' of 5 seconds. You'll also notice that duplicates are allowed, although you can disable duplicate messages within the settings.

$.sticky('<b>Can you see me?</b> I bet you're feeling warm and fuzzy inside right now.');

$.sticky('Upload complete');

$.sticky('<b>You have a new message!</b>');

Returned:

As you can see above, a bit of JSON will be returned with some potentially pertinent information. Now, let's use Sticky in a more traditional sense and attach it to an object. Implementing

Getting sticky up and running is extremely easy. Just add jQuery to your head section along with the Sticky files.

        <script type="text/javascript" src="http://code.jquery.com/jquery-1.6.2.min.js"></script>
        <script type="text/javascript" src="sticky.js"></script>
        <link rel="stylesheet" href="sticky.css" type="text/css" />

Now, you're ready to get busy sending Sticky notes!

         <script>
         $(document).ready(function() {
             $.sticky('The page has loaded!');
         });
         </script>

Or attach it to the object your choice.

        $('#download_status').sticky();

Manipulating

The styling behind Sticky is all CSS. If you want to go for a different look, that's how you'll make it happen. The position can't change from one Sticky to another so that option cannot be passed when calling Sticky. You will have to edit that variable within the sticky.js file to change the position. The other options are pretty straight forward and easy to adjust.

        var position = 'bottom-right';     // top-left, top-right, bottom-left, or bottom-right
        var settings = {
            'speed' : 'fast',   // animations: fast, slow, or integer
            'allowdupes' : true,    // true or false
            'autoclose' : 5000 // integer or false
        };
